MOKSE API Documentation

MOKSE is a simple web API for retrieving scheduling and status information for Eskom Loadshedding, all of which is already publicly available.

Endpoints:

/GetStatus

Retrieve the current Loadshedding stage and the time that the request was made. 

{

    "stage":1,

    "time":"Sat Jan 23 19:12:47 SAST 2021"

}
